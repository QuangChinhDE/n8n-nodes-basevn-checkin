# n8n-nodes-basevn-checkin

This is an n8n community node for BaseVN Checkin API integration.

[n8n](https://n8n.io/) is a [fair-code licensed](https://docs.n8n.io/reference/license/) workflow automation platform.

## Installation

Follow the [installation guide](https://docs.n8n.io/integrations/community-nodes/installation/) in the n8n community nodes documentation.

## Operations

### Cloud Integration
- **Real Time Checkin**: Push real-time checkin log to Base HRM
- **Get User Codes**: Retrieve user checkin codes for cloud integration
- **Mass Sync**: Sync large batches of checkin logs

### Logs
- **Get Logs**: Retrieve checkin logs within a timestamp range

## Credentials

Configure the following credentials:
- **Access Token**: Your Checkin API access token (for ExtAPI operations)
- **Client Token**: Your cloud integration client token (for Cloud operations)

## Compatibility

Tested with n8n version 1.0+
